# Cài đặt

### Cách 1: Community Nodes (khuyến nghị)

1. Mở n8n Editor.
2. Vào **Settings -> Community Nodes**.
3. Chọn **Install** và nhập `n8n-nodes-ultimate`.
4. Nhấn **Install**.
5. Tìm `codedao12` trong danh sách node.

### Cách 2: Tự host (npm)

```bash
cd YOUR_N8N_DIRECTORY
npm install n8n-nodes-ultimate
```

## Lưu ý

- Yêu cầu n8n v0.187 hoặc mới hơn.
- Hoạt động ổn định trên cả n8n V1, V2

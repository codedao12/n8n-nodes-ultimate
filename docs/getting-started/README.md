# Bắt đầu

Tài liệu này hướng dẫn cách cài đặt gói, cấu hình credential, và thử nghiệm nhanh các node.

## Lưu ý quan trọng

- Zalo User và Facebook User là **unofficial**, cần dùng cẩn thận với tài khoản thật.
- Zalo Bot là **official** (API chính thức).
- Khi dùng các tính năng unofficial, nên ưu tiên tài khoản phụ để giảm rủi ro.
- Khuyến nghị n8n v0.187 trở lên để tương thích tốt.

# codedao12 - n8n API

Tài liệu tham khảo: https://docs.n8n.io/api/authentication/

## Parameters

- **API Key (bắt buộc)**: API Key của n8n (Settings → n8n API)
- **Base URL (bắt buộc)**: "http://127.0.0.1:5678" hoặc "https://domain.vn" (URL của n8n instance)

### Lưu ý

- Nếu test connection báo lỗi, hãy thử tạo lại API n8n hoặc xóa bớt API cũ.

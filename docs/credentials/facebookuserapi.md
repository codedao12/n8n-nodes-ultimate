# codedao12 - Facebook User API

Tài liệu tham khảo: https://developers.facebook.com/docs/

## Parameters

- **Cookies User**: cookies Facebook cá nhân và Messenger. Dùng extension trình duyệt lấy cookies (ví dụ: "J2TEAM Cookies", chấp nhận String hoặc Json).
- **Cookies Business**: cookies từ business.facebook.com để tương tác page/group.
- **Proxy URL (Optional)**: proxy để đăng nhập tài khoản (khuyên dùng cố định, không rotate).

### Lưu ý

- Facebook User API là thư viện không chính thức (Unofficial), nên ưu tiên dùng tài khoản phụ để thử nghiệm.
- Cách lấy cookies: mở https://web.facebook.com → dùng J2TEAM Cookies → Export as text → dán JSON vào trường Cookies User hoặc Cookies Business.

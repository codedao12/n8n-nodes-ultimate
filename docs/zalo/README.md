# codedao12 - Zalo

Nhóm Zalo gồm Zalo User, Zalo Trigger và các credential liên quan.

### Tính năng chính

- Gửi tin nhắn, phản ứng, gửi link, danh thiếp, bank card
- Upload và gửi file đính kèm (ảnh, video, file)
- Quản lý bạn bè và tài khoản cá nhân
- Quản lý nhóm, thành viên, ghi chú, link nhóm
- Sticker, Poll, Tag, Catalog / Product
- Hỗ trợ Expression Credential, tự cập nhật credential khi login lại
- Gộp Media Group thành một output và có `filter` giúp xử lý nhanh hơn

### Credentials account

- **Zalo User API Credential**: dùng cho Zalo User và Zalo Trigger.
- **n8n API Credential**: bắt buộc cho đăng nhập QR.

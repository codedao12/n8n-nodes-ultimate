# Product: Lấy Danh sách

Lấy danh sách sản phẩm trong catalog

`Catalog / Product (zBusiness) → Product: Lấy Danh sách`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **Catalog ID (bắt buộc)**: ID của catalog
- **Product List Last Product ID**: lastProductId (nếu có)

### Options

- **Product List Limit**: Số lượng sản phẩm muốn lấy

### Parameters

- **Product List Page**: Trang sản phẩm (bắt đầu từ 0)
- **Product List Version Catalog**: versionCatalog (nếu có)

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

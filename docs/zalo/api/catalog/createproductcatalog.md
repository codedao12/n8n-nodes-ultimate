# Product: Tạo Mới

Tạo sản phẩm mới trong catalog

`Catalog / Product (zBusiness) → Product: Tạo Mới`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **Catalog ID (bắt buộc)**: ID của catalog

### Attachment

- **Product Photo URLs**: Danh sách URL ảnh, phân tách bằng dấu phẩy (tối đa 5)

### Parameters

- **Product Name (bắt buộc)**: Tên sản phẩm
- **Product Price (bắt buộc)**: Giá sản phẩm (string)
- **Product Description (bắt buộc)**: Mô tả sản phẩm

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

# Catalog: Lấy Danh sách

Lấy danh sách catalog

`Catalog / Product (zBusiness) → Catalog: Lấy Danh sách`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **Catalog Last Product ID**: lastProductId (nếu có)

### Options

- **Catalog Limit**: Số lượng catalog muốn lấy

### Parameters

- **Catalog Page**: Trang catalog (bắt đầu từ 0)

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

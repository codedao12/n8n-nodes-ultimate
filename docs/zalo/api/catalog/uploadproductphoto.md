# Product: Upload Ảnh

Upload ảnh sản phẩm để lấy URL

`Catalog / Product (zBusiness) → Product: Upload Ảnh`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Conversation

- **Product Photo Source Type**: Nguồn ảnh sản phẩm

### Attachment

- **Product Photo File Path**: Đường dẫn file ảnh
- **Product Photo File URL**: URL ảnh
- **Product Photo Binary Property**: Tên binary property
- **Product Photo File Name**: Tên file ảnh (tuỳ chọn)

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

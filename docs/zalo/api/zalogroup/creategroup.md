# Tạo Nhóm

Tạo một nhóm mới

`Group Action → Tạo Nhóm`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **Danh sách User ID (bắt buộc)**: Danh sách userId (phân cách bằng dấu phẩy: id1,id2,id3)

### Parameters

- **Tên Nhóm (bắt buộc)**: Tên của nhóm mới Gợi ý: Zalo có hạn chế về số lượng hàng ngày ~10

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

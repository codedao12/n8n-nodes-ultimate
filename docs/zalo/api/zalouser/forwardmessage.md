# Chuyển Tiếp Tin Nhắn

Chuyển tiếp tin nhắn tới nhiều người dùng hoặc nhóm

`User Action → Chuyển Tiếp Tin Nhắn`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Conversation

- **Type**: Chọn loại cuộc trò chuyện cần tương tác (người dùng hoặc nhóm)

### Target

- **Danh sách ID Nhận (bắt buộc)**: Danh sách ID người dùng hoặc nhóm nhận tin nhắn, phân cách bằng dấu phẩy

### Options

- **TTL**: 0 = vĩnh viễn. Thời gian tồn tại của tin nhắn (giây - Time To Live)

### Parameters

- **Nội Dung Tin Nhắn (bắt buộc)**: Nội dung tin nhắn cần chuyển tiếp Gợi ý: Chức năng phù hợp với gửi hàng loạt, giúp chống spam.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

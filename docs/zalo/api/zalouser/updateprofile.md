# Tài Khoản: Cập Nhật Thông Tin

Cập nhật tên, ngày sinh, giới tính của tài khoản

`User Action → Tài Khoản: Cập Nhật Thông Tin`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Parameters

- **Name (bắt buộc)**: Tên hiển thị
- **Date of Birth (bắt buộc)**: Ngày sinh (YYYY-MM-DD)
- **Gender (bắt buộc)**: Giới tính

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

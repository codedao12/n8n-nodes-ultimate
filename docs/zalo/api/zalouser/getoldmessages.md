# Tài Khoản: Lấy Tin Nhắn Đến Cũ

Lấy các tin nhắn cũ hơn trong một cuộc trò chuyện

`User Action → Tài Khoản: Lấy Tin Nhắn Đến Cũ`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Conversation

- **Type**: Chọn loại cuộc trò chuyện cần tương tác (người dùng hoặc nhóm)

### Message

- **Last Message ID**: Lấy các tin nhắn cũ hơn ID này, để trống để lấy tin nhắn mới nhất (chức năng này có thể không hoạt động như mong muốn) Gợi ý: Hoạt động khi trigger tắt (ko online), có vẻ dữ liệu trả về chỉ tương đối kể từ lần online cuối.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

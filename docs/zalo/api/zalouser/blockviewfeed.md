# Chặn/Bỏ Chặn Xem Nhật Ký

Chặn hoặc bỏ chặn xem nhật ký của người dùng

`User Action → Chặn/Bỏ Chặn Xem Nhật Ký`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **User ID (bắt buộc)**: ID của người dùng

### Parameters

- **Chặn Xem Nhật Ký**: Bật để chặn xem nhật ký, tắt để bỏ chặn

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

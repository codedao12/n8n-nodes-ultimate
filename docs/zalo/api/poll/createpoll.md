# Tạo bình chọn

Tạo một bình chọn mới

`Poll → Tạo bình chọn`

### Credentials account

- **Zalo Credentials hoặc Select a custom credentials ID**: dùng credential hoặc session đã lưu để đăng nhập Zalo

### Target

- **ID Nhóm (bắt buộc)**: ID của nhóm để tạo poll (chỉ hoạt động với nhóm)

### Parameters

- **Chủ đề bình chọn (bắt buộc)**: Đặt câu hỏi bình chọn
- **Các lựa chọn**: Thêm các lựa chọn cho bình chọn
- **Các lựa chọn**: Nhập các lựa chọn, mỗi lựa chọn phân tách bằng dấu phẩy
- **Thời hạn bình chọn**: Thời hạn bình chọn (Để trống nếu không có thời hạn)
- **Chọn nhiều phương án (bắt buộc)**: Người tham gia có thể bình chọn nhiều phương án khác nhau
- **Ẩn người bình chọn (bắt buộc)**: Người tham gia không thể thấy người khác bình chọn

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

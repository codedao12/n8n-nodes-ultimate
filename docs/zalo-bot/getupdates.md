# Get Updates

Lấy danh sách sự kiện mới từ Zalo Bot.

Operation: `Get Updates`

### Credentials account

- **Zalo Bot Credential**: token và base URL của Zalo Bot API.

### Options

- **Timeout**: thời gian chờ trước khi trả kết quả (giây).

### Lưu ý

- Timeout chỉ mang tính thử nghiệm, Zalo không đảm bảo trả đủ dữ liệu.

### Hướng dẫn

1. Chọn đúng operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

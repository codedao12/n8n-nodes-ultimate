# Set Webhook

Thiết lập webhook để nhận sự kiện từ Zalo Bot.

Operation: `Set Webhook`

### Credentials account

- **Zalo Bot Credential**: token và base URL của Zalo Bot API.

### Attachment

- **Webhook URL**: URL webhook nhận sự kiện.

### Parameters

- **Secret Token**: mã bảo mật kèm theo webhook.

### Hướng dẫn

1. Chọn đúng operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

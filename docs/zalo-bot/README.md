# codedao12 - Zalo Bot

Bot Zalo gửi tin, soạn tin, gửi ảnh...

### Tính năng chính

- Gửi tin nhắn, ảnh, sticker
- Gửi trạng thái chat action (typing, upload...)
- Quản lý webhook, lấy thông tin bot
- API chính thống từ Zalo Bot Manager

### Credentials account

- **Zalo Bot Credential**: bắt buộc để gọi Zalo Bot API.

### Lưu ý

- Bot Token được tạo từ Zalo Bot Manager.

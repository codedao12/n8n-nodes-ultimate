# Send Photo

Gửi ảnh từ Zalo Bot.

Operation: `Send Photo`

### Credentials account

- **Zalo Bot Credential**: token và base URL của Zalo Bot API.

### Conversation

- **Chat ID**: ID cuộc trò chuyện.

### Message

- **Caption**: mô tả ảnh (tuỳ chọn).

### Attachment

- **Photo URL**: URL ảnh cần gửi.

### Hướng dẫn

1. Chọn đúng operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

# Send Chat Action

Gửi trạng thái hành động chat (typing, upload, record...).

Operation: `Send Chat Action`

### Credentials account

- **Zalo Bot Credential**: token và base URL của Zalo Bot API.

### Conversation

- **Chat ID**: ID cuộc trò chuyện.

### Attachment

- **Action**: trạng thái cần gửi (Typing, Upload Photo, Record Voice...).

### Hướng dẫn

1. Chọn đúng operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

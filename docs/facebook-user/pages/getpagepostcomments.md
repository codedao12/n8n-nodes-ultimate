# Get Page Post Comments

* Lấy bình luận theo Story ID trên page (user).

`Page → Get Page Post Comments`

### Target

- **Story ID (bắt buộc)**: Story ID dùng để đọc comment (VD: Uzpf...=).

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

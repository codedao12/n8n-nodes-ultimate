# Get Group Posts

* Lấy danh sách bài viết trong nhóm (user).

`Group → Get Group Posts`

### Target

- **Group ID (bắt buộc)**: ID nhóm.

### Options

- **Limit**: Số lượng bài viết cần lấy (1-50).
- **Cursor**: Cursor để phân trang (optional). Gợi ý: truyền end_cursor ở lần lấy trước vào để lấy trang kế tiếp
- **Sorting**: Cách sắp xếp bài viết.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.

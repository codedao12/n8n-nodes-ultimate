# Forward Attachment

Chuyển tiếp attachment theo message ID (user).

`Message → Forward Attachment`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Attachment

- **Forwarded Message ID (bắt buộc)**: ID tin nhắn chứa attachment cần chuyển tiếp.

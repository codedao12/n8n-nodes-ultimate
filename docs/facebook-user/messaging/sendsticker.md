# Send Sticker

Gửi sticker (user).

`Message → Send Sticker`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Sticker

- **Sticker ID (bắt buộc)**: ID sticker cần gửi.

# Edit Message

Sửa nội dung tin nhắn (user).

`Message → Edit Message`

### Message

- **Message ID (bắt buộc)**: ID tin nhắn cần sửa.
- **New Message (bắt buộc)**: nội dung mới.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

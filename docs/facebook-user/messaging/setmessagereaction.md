# Set Message Reaction

Thả cảm xúc (user).

`Message → Set Message Reaction`

### Reaction

- **Message ID (bắt buộc)**: ID của tin nhắn.
- **Reaction**: emoji phản ứng (để trống = xóa reaction).

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

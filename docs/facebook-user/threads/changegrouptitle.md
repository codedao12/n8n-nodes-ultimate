# Thread: Change Title

Đổi tên nhóm (messenger user).

`Thread → Thread: Change Title`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Parameters

- **Group Title (bắt buộc)**: Tên nhóm mới.

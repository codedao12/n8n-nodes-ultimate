# Get Thread History

Lịch sử tin nhắn.

`Thread → Get Thread History`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Options

- **Before Timestamp (ms)**: Mốc thời gian lấy trước đó. 0 = mới nhất.

### Parameters

- **Amount (bắt buộc)**: Số lượng tin nhắn cần lấy.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

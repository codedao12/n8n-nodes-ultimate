# Thread: Change Image

Đổi ảnh nhóm (messenger user).

`Thread → Thread: Change Image`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Attachment

- **Group Image Source**: Chọn nguồn ảnh nhóm.
- **Binary Property Name**: Tên binary property chứa ảnh.
- **File Path**: Đường dẫn file ảnh.
- **Image URL**: URL ảnh.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

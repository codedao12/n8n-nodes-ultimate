# Thread: Add User

Thêm thành viên vào nhóm chat (messenger user).

`Thread → Thread: Add User`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Target

- **User ID(s) (bắt buộc)**: User ID cần thêm/xóa (có thể nhiều, cách nhau bằng dấu phẩy).

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

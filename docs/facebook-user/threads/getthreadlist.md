# Get Thread List

Danh sách hội thoại.

`Thread → Get Thread List`

### Options

- **Limit (bắt buộc)**: Số lượng thread cần lấy.
- **Timestamp (ms)**: Mốc thời gian để phân trang. 0 = mới nhất.

### Parameters

- **Tags**: Bộ lọc thread.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

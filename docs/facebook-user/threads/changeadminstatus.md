# Thread: Change Admin

Thêm/bớt admin nhóm (messenger user).

`Thread → Thread: Change Admin`

### Conversation

- **Thread ID (bắt buộc)**: ID của người dùng hoặc nhóm chat.

### Target

- **Admin User ID(s) (bắt buộc)**: User ID cần đổi quyền admin (có thể nhiều, cách nhau bằng dấu phẩy).

### Parameters

- **Set As Admin**: Bật để set admin, tắt để gỡ admin.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

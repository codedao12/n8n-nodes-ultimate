# Thread: Create New

Tạo nhóm chat mới (messenger user).

`Thread → Thread: Create New`

### Target

- **Participant ID(s) (bắt buộc)**: Danh sách ID tạo nhóm (có thể nhiều, cách nhau bằng dấu phẩy).

### Parameters

- **Group Title**: Tên nhóm (optional).

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

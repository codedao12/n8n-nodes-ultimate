# Recreate Note

Xóa rồi tạo lại.

`Notes → Recreate Note`

### Target

- **Old Note ID (bắt buộc)**: ID note cũ cần xóa.

### Message

- **New Note Text (bắt buộc)**: Nội dung note mới.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

# codedao12 - Facebook User

Tự động hóa Facebook cá nhân (unofficial): nhắn tin, quản lý bạn bè, nhóm chat, đăng bài và nhiều tiện ích khác.

### Tính năng chính

- Nhắn tin, reaction, gửi sticker, forward attachment
- Quản lý bạn bè, follow/unfollow, friend requests
- Quản lý thread/group chat
- Đăng bài, comment, share preview
- Trang, nhóm, notes, story, sticker, upload media

### Credentials account

- **Facebook User API Credential**: dùng để đăng nhập tài khoản Facebook.

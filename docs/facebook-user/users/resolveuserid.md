# Resolve Name To User ID

Lấy User ID từ Name hoặc Link Facebook.

`User → Resolve Name To User ID`

### Target

- **Name or Link (bắt buộc)**: Name hoặc link Facebook để lấy ID. Gợi ý: nút này sử dụng API từ id.traodoisub.com

2. Nhập các trường cần thiết theo mô tả.

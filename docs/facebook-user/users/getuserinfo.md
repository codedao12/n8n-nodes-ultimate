# Get User Info

Lấy thông tin người dùng từ User ID.

`User → Get User Info`

### Target

- **User ID (bắt buộc)**: ID/vanity/URL người dùng cần lấy info.

### Parameters

- **Use Payload**: Bật để dùng API chat/user_info (nhanh và ít tài nguyên, có thể sai giới tính). Gợi ý: bật usePayload sẽ có kết quả nhanh hơn và có thể sai giới tính

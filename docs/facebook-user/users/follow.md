# Follow / Unfollow

Theo dõi hoặc bỏ theo dõi.

`User → Follow / Unfollow`

### Target

- **User ID (bắt buộc)**: ID người dùng.

### Options

- **Follow**: Bật để follow, tắt để unfollow.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

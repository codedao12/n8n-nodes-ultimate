# Create Story

Tạo story.

`Story → Create Story`

### Message

- **Story Text (bắt buộc)**: Nội dung story.
- **Font**: Font cho story.
- **Background**: Nền story.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

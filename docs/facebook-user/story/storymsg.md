# Reply Story

Trả lời story.

`Story → Reply Story`

### Target

- **Story ID or URL (bắt buộc)**: ID hoặc URL của story.

### Message

- **Story Reply Message (bắt buộc)**: Nội dung trả lời story.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.

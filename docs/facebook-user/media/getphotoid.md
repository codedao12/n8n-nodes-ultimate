# Get Photo ID

Upload ảnh để lấy Photo ID.

`Upload Media → Get Photo ID`

### Conversation

- **Attachment Type**: Chọn cách upload media. Gợi ý: Upload media: chọn Binary/File Path/URLs để lấy Photo ID.

### Target

- **Personal or Page ID**: ID người đăng hoặc pageId đăng Gợi ý: Leave blank if using a personal account or enter page ID.

### Attachment

- **Binary Property Name(s)**: Tên binary property (có thể nhiều, cách nhau bằng dấu phẩy).
- **File Path(s)**: Đường dẫn file (có thể nhiều, cách nhau bằng dấu phẩy).
- **Attachment URL(s)**: URL ảnh (có thể nhiều, cách nhau bằng dấu phẩy).

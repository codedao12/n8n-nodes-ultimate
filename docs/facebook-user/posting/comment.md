# Post: Comment

Bình luận bài viết bất kì (user, fb.com/groups/codedao12/posts/..ID/)

`Post → Post: Comment`

### Target

- **Post ID (bắt buộc)**: ID bài viết (chuỗi số).
- **Reply Comment ID**: ID comment cần reply (optional).

### Message

- **Comment Text (bắt buộc)**: Nội dung comment.

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.

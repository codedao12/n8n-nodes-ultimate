# Post: Create (Group)

* Đăng bài lên nhóm (user).

`Post → Post: Create (Group)`

### Conversation

- **Attachment Type**: Chọn cách đính kèm ảnh.

### Target

- **Group ID (bắt buộc)**: ID nhóm.
- **Photo ID(s)**: ID ảnh đã upload (có thể nhiều, cách nhau bằng dấu phẩy). Gợi ý: Sử dụng nút Upload Media để lấy photoID.
- **Doc ID Override**: Để trống để sử dụng mặc định.

### Message

- **Post Text**: Nội dung bài viết.

### Attachment

- **Binary Property Name(s)**: Tên binary property (có thể nhiều, cách nhau bằng dấu phẩy).
- **File Path(s)**: Đường dẫn file (có thể nhiều, cách nhau bằng dấu phẩy).
- **Attachment URL(s)**: URL ảnh (có thể nhiều, cách nhau bằng dấu phẩy).

### Options

- **Debug Raw Response**: Bật để trả về raw response khi debug.

### Hướng dẫn

1. Chọn đúng resource và operation.
2. Nhập các trường cần thiết theo mô tả.
3. Test node với dữ liệu mẫu để kiểm tra output.
